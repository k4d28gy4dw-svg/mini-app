# Классические игры — Telegram Mini App

Мини-приложение с шашками, шахматами и морским боем. Каждая игра поддерживает режим против бота, онлайн-комнаты для двух игроков и чат внутри комнаты.

## Игры

- Шашки с обязательным взятием, сериями взятий и тремя уровнями бота.
- Шахматы с ходами всех фигур, превращением пешки и ботом.
- Морской бой с автоматической расстановкой полного флота и ботом.

## Локальный запуск

Проект не требует сборки. Запустите статический сервер в папке проекта, например:

```bash
python3 -m http.server 8080
```

Затем откройте `http://localhost:8080`. Режим с ботом работает без Supabase.

## Настройка онлайн-режима

1. Создайте проект в Supabase.
2. Откройте SQL Editor и выполните файл `supabase.sql`.
3. Включите Realtime для таблицы `checkers_rooms`, если команда в конце SQL-файла сообщает, что таблица уже добавлена в публикацию — это нормально.
4. В Project Settings → API скопируйте Project URL и anon/public key.
5. Вставьте значения в `config.js`. Не используйте `service_role` key в браузере.

## Подключение к Telegram

1. Опубликуйте файлы на HTTPS-хостинге, например GitHub Pages, Netlify или Vercel.
2. Откройте `@BotFather`, выберите бота и укажите опубликованный URL через настройку Menu Button или Mini App.
3. Откройте приложение из меню бота и создайте комнату. Ссылку на комнату можно отправить второму игроку.

`config.js` содержит только публичный anon key. Доступ к таблице ограничивается политиками RLS из `supabase.sql`.
